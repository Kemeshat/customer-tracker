# Managing Customer Data with Arrays and Objects

## Project Description
This project is a simple customer management system built using JavaScript. It stores customer information such as names, emails, and purchase history using arrays and objects. The goal is to simulate how a small business might keep track of its clients and their activity.

## Technologies Used
- JavaScript
- HTML

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open the `index.html` file in a web browser.
4. Open the browser console to view the output.

## What I Learned
While working on this project, I learned how to use arrays and objects together to organize and manage data. I practiced using array methods like push() and shift() to add and remove data, and I also learned how to update specific values inside an object. 

I also improved my understanding of looping through data using forEach() and displaying information in a clear way using template literals. Overall, this project helped me better understand how JavaScript can be used to manage real-world data in a structured way.
